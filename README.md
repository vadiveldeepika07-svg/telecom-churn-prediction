# Telecom Churn Prediction

## 📌 Project Overview
This project predicts whether a customer will churn using machine learning techniques.

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Excel

## 🔍 Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Building (Logistic Regression, Decision Tree)
- Model Evaluation

## 📊 Result
Achieved good accuracy in predicting customer churn.

## 🚀 Future Improvements
- Try advanced models like Random Forest, XGBoost
